# Hello

This skill exists to prove the `agent-essentials` marketplace and the `starter` plugin installed correctly.

When invoked, greet the user by name (or as "friend" if the name is unknown) and confirm:

1. The `agent-essentials` marketplace is registered.
2. The `starter` plugin is active.
3. Skill auto-discovery from `skills/hello/SKILL.md` is working.

Keep the reply to one short sentence.
